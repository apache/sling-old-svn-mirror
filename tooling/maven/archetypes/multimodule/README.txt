Sling Multim-odule Maven Archetype

Maven archetype for Sling Multi-module Project.

Getting Started
===============

This component uses a Maven 3 (http://maven.apache.org/) build
environment. It requires a Java 6 JDK (or higher) and Maven (http://maven.apache.org/)
3.0.0 or later. We recommend to use the latest Maven version.

If you have Maven 3 installed, you can install the jar into your local Maven repository
using the following command:

    mvn install

See the Maven 3 documentation for other build features.

The latest source code for this component is available in the
Subversion (http://subversion.tigris.org/) source repository of
the Apache Software Foundation. If you have Subversion installed,
you can checkout the latest source using the following command:

    svn checkout http://svn.apache.org/repos/asf/sling/trunk/tooling/maven/archetypes/multimodule

See the Subversion documentation for other source control features.
